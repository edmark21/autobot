# autobot
helps you to identify phone numbers, the name associated with that number and even their addresses. 

How to use?

Installation

$ git clone https://github.com/edmark21/autobot

$ cd autobot

$ pip3 install twilio

(run the program by typing:)

(version 1)
$ python3 search.py

or

(version 2)
$ python3 search2.py
